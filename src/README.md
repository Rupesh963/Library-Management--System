# Library Management System

## Description
A simple Java Console-based Library Management System developed using Object-Oriented Programming (OOP) concepts. This application allows users to add books, view available books, issue books, and return books through a menu-driven interface.

## Features
- Add Book
- View Books
- Issue Book
- Return Book
- Menu Driven Console Application

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Objects
- Encapsulation
- ArrayList
- Methods
- Loops
- Switch Case
