# Library Management System

## Description
A simple Java Console-based Library Management System developed using Object-Oriented Programming (OOP) concepts. This application allows users to add books, view available books, issue books, and return books through a menu-driven interface.

## Features
- Add Book
- View Books
- Issue Book
- Return Book
- Menu-Driven Console Application

## Technologies Used
- Java
- IntelliJ IDEA
- ArrayList
- Git & GitHub

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Objects
- Encapsulation
- ArrayList
- Methods
- Loops
- Switch Case

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run the `Main.java` file.
3. Select an option from the menu.
4. Perform library operations.

## Sample Menu
